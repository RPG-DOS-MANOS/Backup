Compendium Folders are now in core Foundry! This means that there isn't a need for this module anymore, as core Foundry can take care of compendium organization. As a result I don't plan on making any more updates to Compendium Folders

I've included some migration code so all your folders both in the sidebar and inside compendiums will be converted to core Foundry folders.

The sidebar migration will begin once you click the button below. To migrate compendiums with folders in them, there's a new module setting called Migrate Compendiums.

Note: it is strongly recommended to back up your compendiums before running the migration. If you run into any issues with the migrations feel free to raise an issue on the GitHub or ping me on discord
